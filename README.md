# G.R. Minds Technologies

Welcome to **G.R. Minds Technologies** – transforming ideas into digital reality through innovation, excellence, and scalable technology solutions. This is a modern, interactive website built using **React**, **Framer Motion**, and **Tailwind CSS**, designed to showcase services, team, and solutions for IT businesses.

---

## 🚀 Features

- **Hero Section**: Animated 3D canvas with parallax effects and interactive call-to-action buttons.  
- **Responsive Navbar**: Modern, sticky header with mobile-friendly menu and smooth scrolling.  
- **Team Section**: Showcases Founder & Co-Founder with experience, skills, and descriptions.  
- **Services Section**: Highlighted IT services including web/mobile development, cloud solutions, cybersecurity, bulk messaging, and more.  
- **About Section**: Mission, values, stats, and innovation highlights with animations.  
- **Footer**: Modern footer with contact info, social media links, services, and company info.  
- **Particle Background**: Subtle, animated particle background for a modern tech feel.  
- **Mobile-Friendly**: Fully responsive design optimized for desktop, tablet, and mobile devices.  
- **Animations**: Smooth transitions, scroll reveal effects, and hover interactions using Framer Motion.  

---

## 🛠️ Technologies Used

- **React** – UI library for building components.  
- **TypeScript** – Type-safe JavaScript for scalable development.  
- **Tailwind CSS** – Utility-first CSS framework for modern design.  
- **Framer Motion** – Animation library for interactive UI elements.  
- **React Three Fiber & Drei** – 3D graphics and interactive canvas elements.  
- **Lucide Icons** – Modern, customizable icons.
